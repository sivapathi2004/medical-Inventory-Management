Naan Mudhalvan Project – Medical Inventory Management
Team: We Team | Department: CSE-B | Batch: Final Year (2026)
Project Overview :

This project is developed as part of the Naan Mudhalvan Initiative under the Salesforce domain.
Our objective is to design and implement a custom Salesforce application that demonstrates practical CRM functionalities such as data management, automation, and analytics.

Project Objective :

To create an efficient Salesforce-based business application that simplifies customer and business process management through:

Automated workflows

Data visualization with dashboards

Customized objects and relationships

Reports for business insights

Features Implemented :

Custom Objects: Designed to store domain-specific records (e.g., Orders, Products, Customers).

Relationships: Implemented Master-Detail and Lookup relationships for structured data management.

Automation: Configured Workflow Rules, Process Builder, and Validation Rules for automatic updates and data consistency.

Reports & Dashboards: Created analytical reports for key performance insights.

User Interface: Customized page layouts, tabs, and record pages for better user experience.

Technologies Used :

Platform: Salesforce Lightning

Tools: Flow Builder, Process Builder, Reports, Dashboards

Languages: Apex (for logic) & SOQL (for queries)

Environment: Salesforce Developer Org / Trailhead Playground

Modules in the Project :

Customer Management – Handles customer records and interactions

Order Management – Tracks orders and product details

Inventory Management – Manages stock and availability

Reports & Dashboards – Displays key data insights
